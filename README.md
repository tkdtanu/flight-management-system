# Flight Management System

Business Operation
----------
1. Search Flights for given source, destination and Date
2. Search Flights for given Airline

Technology Used
---------------
1. Java
2. Spring Boot
3. Spring Data JPA
4. H2 DB (In Memory)
5. SQL
6. Hibernate

Entities
--------
1. Airport
2. Airline
3. Flight
4. Seat
5. FlightScehdule
6. FlightScehduleSeat

Note
